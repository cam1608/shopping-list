This repository contains code for a simple todo list web application. Users can add new tasks, mark tasks as completed by clicking on them, and delete tasks individually.

## Features

- Allows users to add new tasks to the todo list.
- Supports marking tasks as completed by clicking on them.
- Provides an option to delete individual tasks.
- Supports both mouse click and keyboard input for adding new tasks.
- Dynamically adds delete buttons to each task for easy removal.

## Usage

1. Open the `index.html` file in a web browser.
2. Enter a new task in the input field.
3. Press the "Enter" key or click the "Enter" button to add the task to the list.
4. Click on a task to mark it as completed. Click again to undo.
5. Click on the "Delete" button next to a task to remove it from the list.
